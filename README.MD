# NoSQL Notes & Activity Tracker

## Overview

This project is a web application designed to demonstrate the use of a NoSQL database (MongoDB) for managing flexible and scalable data. The system allows users to create notes, add comments, assign tags, and track activity such as views and edits.

The project focuses on highlighting the advantages of NoSQL over traditional relational databases in terms of schema flexibility, performance, and scalability.

## Features

* Create, read, update, and delete notes
* Add comments to notes
* Assign tags to notes
* Track note views and activity
* Search and filter notes by tags

## Tech Stack

* Frontend: Next.js (React framework)
* Backend: NestJS (Node.js framework)
* Database: MongoDB (NoSQL)

## Getting Started

### Prerequisites

* Node.js
* MongoDB

### Installation

1. Clone the repository:

   ```
   git clone <repo-url>
   ```

2. Install dependencies:

   ```
   cd backend
   npm install
   cd ../frontend
   npm install
   ```

3. Run the application:
   Backend:

   ```
   npm run dev
   ```

   Frontend:

   ```
   npm start
   ```
